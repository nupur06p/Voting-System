# Voting-System
Voting System can be useful for conducting voting at the organizational level as well as at the national level.

The details of the person voting should be secure in the blockchian system and their privacy should be maintained.
Voter should be eligible to vote and the eligibility of the voter should be checked before casting of the vote. 
User can vote only once and the system should prevent teh voter from casting anotehr vote.
System should abide by the rules of voting and teh consitions should be transparent.
The number of votes should be accurately recorded and counted.
There should not be any kind of fraud or any possibility of mistake in the voting system.
